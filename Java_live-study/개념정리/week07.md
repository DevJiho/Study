## `package`

> 클래스와 인터페이스의 집합.

- 클래스들을 하나로 묶어놓은 것
- 클래스 간의 이름 중복으로 발생하는 충돌을 막아준다.
- 클래스를 기능별로 분류할 수 있어 식별에 용이해다.

**패키지는 해당 이름을 가진 디렉토리를 의미**한다.


`선언방식`
~~~java

package 패키지이름;

~~~

## import 

코드에서 사용할 클래스의 패키지에 대한 정보를 미리 제공하는 역할.

`선언방식
~~~java

import 패키지이름.클래스이름;

~~~

클래스 이름자리에 * (와일드카드) 를 이용하여 해당 패키지내의 모든 클래스를 사용할 수 있다.
